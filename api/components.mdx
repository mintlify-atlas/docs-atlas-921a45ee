## Overview

Discord message components are interactive elements that can be added to messages. This page documents all available component type definitions.

## MessageComponent Union Type

```typescript
type MessageComponent =
  | ActionRow
  | Button
  | StringSelect
  | UserSelect
  | RoleSelect
  | MentionableSelect
  | ChannelSelect
  | TextInput
  | Section
  | TextDisplay
  | Thumbnail
  | MediaGallery
  | FileComponent
  | Separator
  | Container
  | Label;
```

---

## Interactive Components

### Button

Interactive button component that users can click.

```typescript
type Button = CustomButton | LinkButton | PremiumButton;

interface CustomButton extends LabeledButton {
  custom_id: string;
  style:
    | ButtonStyleTypes.PRIMARY
    | ButtonStyleTypes.SECONDARY
    | ButtonStyleTypes.SUCCESS
    | ButtonStyleTypes.DANGER;
}

interface LinkButton extends LabeledButton {
  url: string;
  style: ButtonStyleTypes.LINK;
}

interface PremiumButton extends BaseButton {
  sku_id: string;
  style: ButtonStyleTypes.PREMIUM;
}

interface LabeledButton extends BaseButton {
  emoji?: Pick<EmojiInfo, 'id' | 'name' | 'animated'>;
  label: string;
}

interface BaseButton extends BaseComponent {
  disabled?: boolean;
  type: MessageComponentTypes.BUTTON;
}
```

**Example:**

```typescript
const button: Button = {
  type: MessageComponentTypes.BUTTON,
  style: ButtonStyleTypes.PRIMARY,
  label: 'Click me',
  custom_id: 'my_button',
  disabled: false,
};
```

**Reference:** [Discord Button Documentation](https://discord.com/developers/docs/components/reference#button)

### ActionRow

Container component for organizing interactive components.

```typescript
type ActionRow = BaseComponent & {
  type: MessageComponentTypes.ACTION_ROW;
  components: Array<
    | Button
    | StringSelect
    | UserSelect
    | RoleSelect
    | MentionableSelect
    | ChannelSelect
    | TextInput
  >;
};
```

**Example:**

```typescript
const actionRow: ActionRow = {
  type: MessageComponentTypes.ACTION_ROW,
  components: [
    {
      type: MessageComponentTypes.BUTTON,
      style: ButtonStyleTypes.PRIMARY,
      label: 'Button 1',
      custom_id: 'button_1',
    },
    {
      type: MessageComponentTypes.BUTTON,
      style: ButtonStyleTypes.SECONDARY,
      label: 'Button 2',
      custom_id: 'button_2',
    },
  ],
};
```

**Reference:** [Discord Action Row Documentation](https://discord.com/developers/docs/components/reference#action-row)

## Select Menus

### StringSelect

Dropdown menu for selecting from predefined text options.

```typescript
type StringSelect = SelectMenu<MessageComponentTypes.STRING_SELECT> & {
  options: StringSelectOption[];
};

type StringSelectOption = {
  label: string;
  value: string;
  description?: string;
  emoji?: Pick<EmojiInfo, 'id' | 'name' | 'animated'>;
  default?: boolean;
};

type SelectMenu<T extends SelectComponentType> = BaseComponent & {
  type: T;
  custom_id: string;
  placeholder?: string;
  min_values?: number;
  max_values?: number;
  disabled?: boolean;
  required?: boolean;
};
```

**Example:**

```typescript
const stringSelect: StringSelect = {
  type: MessageComponentTypes.STRING_SELECT,
  custom_id: 'color_select',
  placeholder: 'Choose a color',
  min_values: 1,
  max_values: 1,
  options: [
    {
      label: 'Red',
      value: 'red',
      description: 'The color red',
      emoji: { name: '🔴' },
    },
    {
      label: 'Blue',
      value: 'blue',
      description: 'The color blue',
      emoji: { name: '🔵' },
    },
  ],
};
```

**Reference:** [Discord String Select Documentation](https://discord.com/developers/docs/components/reference#string-select)

enum ChannelTypes {

## Display Components

### Section

Section component for grouping related content with an optional accessory.

```typescript
interface Section extends BaseComponent {
  type: MessageComponentTypes.SECTION;
  components: TextDisplay[] & { length: 1 | 2 | 3 };
  accessory: Thumbnail | Button;
}
```

**Example:**

```typescript
const section: Section = {
  type: MessageComponentTypes.SECTION,
  components: [
    {
      type: MessageComponentTypes.TEXT_DISPLAY,
      content: 'This is a section with text',
    },
  ],
  accessory: {
    type: MessageComponentTypes.THUMBNAIL,
    media: {
      url: 'https://example.com/image.png',
    },
  },
};
```

**Reference:** [Discord Section Documentation](https://discord.com/developers/docs/components/reference#section)

## Base Types

### BaseComponent

Base interface that all components extend.

```typescript
interface BaseComponent {
  type: MessageComponentTypes;
  id?: number;
}
```

### EmojiInfo

Type definition for emoji objects used in components.

```typescript
type EmojiInfo = {
  name: string | undefined;
  id: string | undefined;
  user?: { [key: string]: any };
  roles?: string[];
  require_colons?: boolean;
  managed?: boolean;
  available?: boolean;
  animated?: boolean;
};
```

## Complete Example

Here's a complete example combining multiple components:

```typescript
import {
  MessageComponentTypes,
  ButtonStyleTypes,
  TextStyleTypes,
} from 'discord.js';

const message = {
  content: 'Choose your preferences:',
  components: [
    {
      type: MessageComponentTypes.ACTION_ROW,
      components: [
        {
          type: MessageComponentTypes.STRING_SELECT,
          custom_id: 'preference_select',
          placeholder: 'Select your preference',
          options: [
            { label: 'Option A', value: 'a', emoji: { name: '🅰️' } },
            { label: 'Option B', value: 'b', emoji: { name: '🅱️' } },
            { label: 'Option C', value: 'c', emoji: { name: '©️' } },
          ],
        },
      ],
    },
    {
      type: MessageComponentTypes.ACTION_ROW,
      components: [
        {
          type: MessageComponentTypes.BUTTON,
          style: ButtonStyleTypes.SUCCESS,
          label: 'Confirm',
          custom_id: 'confirm_button',
        },
        {
          type: MessageComponentTypes.BUTTON,
          style: ButtonStyleTypes.DANGER,
          label: 'Cancel',
          custom_id: 'cancel_button',
        },
      ],
    },
  ],
};
```
